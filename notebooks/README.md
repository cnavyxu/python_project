# Jupyter Notebooks

本目录包含各种演示和教程的 Jupyter notebooks。

## Notebooks 列表

- `01_sorting_algorithms.ipynb` - 排序算法演示
- `02_machine_learning_basics.ipynb` - 机器学习基础
- `03_neural_networks.ipynb` - 神经网络入门
- `04_data_structures.ipynb` - 数据结构使用

## 使用方法

1. 启动 Jupyter：
   ```bash
   jupyter notebook
   ```

2. 在浏览器中打开相应的 notebook

3. 运行单元格来查看示例和结果

## 环境要求

确保已安装所有必要的依赖：

```bash
pip install -r requirements-dev.txt
```
