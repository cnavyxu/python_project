# 数据目录

本目录用于存储项目使用的数据文件。

## 目录结构

```
data/
├── raw/           # 原始数据
├── processed/     # 处理后的数据
└── samples/       # 示例数据
```

## 说明

### raw/
存放原始的、未经处理的数据文件。

### processed/
存放经过预处理、清洗或转换后的数据文件。

### samples/
存放用于测试和演示的小型示例数据集。

## 注意事项

- 数据文件不会被提交到 Git 仓库（已在 .gitignore 中配置）
- 大型数据集应使用外部存储或数据管理工具
- 建议在项目文档中说明如何获取和准备数据

## 数据格式

推荐使用以下格式：

- CSV: 表格数据
- JSON: 结构化数据
- NPY/NPZ: NumPy 数组
- HDF5: 大型数据集
- Pickle: Python 对象序列化
